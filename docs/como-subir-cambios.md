# Cómo subir cambios (paso a paso)

Seguí esto SIEMPRE:

---

## 1. Abrir terminal en el proyecto

---

## 2. Traer cambios del repo

```bash
git pull
```

---

## 3. Ver qué cambiaste

```bash
git status
```

---

## 4. Agregar cambios

```bash
git add .
```

---

## 5. Crear commit

```bash
git commit -m "descripción corta"
```

Ejemplo:

```bash
git commit -m "agrego ejercicio 2"
```

---

## 6. Subir cambios

```bash
git push
```

---

## ✔️ Listo

Tus cambios ya están en GitHub.
